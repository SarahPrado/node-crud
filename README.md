# node-crud

Criar arquivo server.js

```js
npm init -y
```

Instalar o nodemon

```js
npm i nodemon
```

No package.json

```json
"start": "nodemon serve.js"
```

Adicionar o module
```json
"type": "module"
```

criar o .gitignore e adicionar a pasta *node_modules* para nao ser enviado para o repositorio remoto
```
node_modules
```
